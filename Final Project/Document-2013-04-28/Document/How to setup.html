Steps for installing the sample application

Follow the steps given below to install the sample application:
(for more information about any step from steps 2-4, consult the user manual).

  1. Install JRE\JDK, the Web Server(Apache Tomcat), and the database software(DB2 - Express 'C').
  2. Create the database for the application (the name of database is CMC). The database should be created in the user account "db2admin" and the password for the account should be "admin". It is required that you check Enable XML for Database.
  3. Deploy the application on the webserver.
  4. The database backup has been provided in the "Database" folder.
  5. After the database has been created, the application is ready to be executed. Open a web-browser and go to the URL http://localhost:8080/cmc/
